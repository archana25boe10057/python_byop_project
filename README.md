# Student Health Tracker (Python CLI)

## Overview  
This is a command-line based health tracking application designed to help students monitor their daily lifestyle and health habits. It provides a simple and structured way to log, analyze, and improve overall well-being.

## Problem Statement  
Many students do not consistently track their health metrics such as sleep, diet, or activity levels, which can lead to unhealthy routines. This project aims to solve that problem by providing an easy-to-use tool for tracking and gaining insights into daily health patterns.

## Features  
- Daily health logging (sleep, water, screen time, steps, mood, calories)  
- Weekly health score calculation  
- Personalized health recommendations  
- Calorie tracker (meals, exercise, goals, summaries)  
- Period tracker (cycle logging, prediction, symptoms tracking)  
- Data visualization using graphs  
- Data stored persistently using JSON files
